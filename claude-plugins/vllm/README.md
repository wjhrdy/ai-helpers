# vLLM Plugin

Tools and skills specific for vLLM.

## Skills

### `vllm-slack-summary`

Generate message summaries of vLLM CI SIG Slack channel activity for the RHAIIS midstream release team. Automates extracting channel history via slackdump and creating structured reports focused on breaking changes, hardware issues, CI/CD infrastructure changes, and action items.

## Installation

```bash
/plugin install vllm@odh-ai-helpers
```
